# NotTooLate Bot — Спецификация кнопок

## Раскладка главного меню (ReplyKeyboard)

```
[         ➕ Добавить дату           ]
[ 📋 Все даты    ] [ ✏️ Редактировать ]
[ 🗑 Удалить     ] [ ❓ Помощь        ]
```

## Поведение кнопок

### 1. ➕ Добавить дату

Пошаговое добавление даты:

1. Выбор типа (день рождения, годовщина, другое)
2. Ввод имени (пресеты или вручную)
3. Выбор даты через календарь

### 2. 📋 Все даты

- Показать список всех дат, отсортированных по ближайшим
- Под каждой датой — inline-кнопки: "✏️ Редактировать" и "🗑 Удалить"
- "Редактировать" — показать выбор поля: Название, Дата
- "Удалить" — запросить подтверждение, при "Да" удалить

### 3. ✏️ Редактировать

- Показать список дат для выбора (inline-кнопки)
- После выбора даты — показать варианты: Название, Дата
- Название — ввод нового текста
- Дата — выбор через календарь

### 4. 🗑 Удалить

- Показать список дат для выбора (inline-кнопки)
- После выбора — запросить подтверждение ("Да, удалить" / "Нет")
- При подтверждении — удалить дату

### 5. ❓ Помощь

- Показать справку по всем командам и кнопкам
